# Project Name: MediROUT
A desktop application built with C# and .NET using Three-Tier Architecture.

## Features
##  Technical Implementation
  * Three-Tier Architecture: Clean separation of concerns between Presentation, Business Logic, and Data Access layers.
  * Advanced Data Querying: Extensive use of LINQ for efficient data manipulation and retrieval.
  * Design Patterns: Implementation of industry-standard patterns to ensure code scalability and maintainability.
  * Data Serialization: Integrated XML for data storage and configuration management.
  * Database Integration: Full integration with a Relational Database for persistent data management.
  * User-friendly UI: Designed with a focus on intuitive user experience.
  * Version Control: Developed using Git for source code management and documentation
##  System Capabilities
    Administrator Portal:
    Dashboard: A central view for system information and status.
    Management: Full control over couriers and orders, including adding new entries with detailed profiles.
    Database Maintenance: Tools to initialize the database with default values or perform a complete data reset/wipe.
    
    Courier Portal:
    Profile Management: View and dynamically update personal courier information.
    Delivery Tracking: A dedicated history window summarizing all completed shipments.
    Smart Dispatching: Option to select new orders based on a customizable delivery distance radius.

## Tech Stack
* Language: C#
* Framework: .NET
* Tools: Git, SQL, XML, LINQ
